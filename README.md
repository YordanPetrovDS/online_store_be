# Online Store API

## Installation

#### Install Python 3:

- Please checkout https://realpython.com/installing-python/

#### Create virtual environment:

- Go to you project directory, then

```shell
> py -3 -m venv .venv
```

```shell
> .\.venv\Scripts\Activate.ps1
```

#### Install requirements:

```shell
> pip install -r requirements.txt
```
